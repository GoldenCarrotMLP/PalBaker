# What is Altermatic?

Altermatic is a revolutionary, dynamic variant swap framework developed by the Palworld modding community. 

Traditionally, mesh mods are "replacers"—meaning if you replace Chillet's mesh, every single Chillet spawning in your world will look identical. Altermatic completely removes this limitation.

---

## Dynamic Runtime Variant Swapping

With Altermatic active, your custom models are registered as **dynamic variants**:
- You can create multiple different skins, accessories, or completely different meshes for a single Pal.
- These models are loaded dynamically in memory at runtime.
- A single species can spawn in the wild with rich visual variety!

---

## Variant Swap Parameters

PalBaker's Altermatic Configurator allows you to customize exact spawn parameters for each variant:

- **Gender Constraints**: Force a model to only spawn on Male or Female instances of the Pal.
- **Lucky / Giant Attributes**: Force a variant to only trigger when a Pal spawns as a "Lucky" (giant/shiny) overworld encounter.
- **Passive Trait Matching**: Force a skin to appear only if the Pal possess a specific passive trait (such as showing a gold skin only if the Pal is "Artisan").
