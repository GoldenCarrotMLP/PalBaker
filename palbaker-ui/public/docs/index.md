# PalBaker User Guide
Welcome to the official PalBaker documentation! PalBaker is a powerful suite designed to make Palworld modding automated, visual, and pain-free. 

Select a topic from the sidebar on the left to learn how to set up your environment, cook models, replace audio, and create brand-new standalone Pals!
