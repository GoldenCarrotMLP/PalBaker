## Durable Decisions
- **Modularity Limits**: Any file exceeding 350-400 lines must be split into dedicated domain utility handlers.
- **Tracebacks to UI**: Intercept raw Python tracebacks, pass them to a central matcher, and map them to friendly Kemono illustrations/dialogues with high-utility recovery buttons rather than raw text banners.
- **No Hardcoded Release Filenames**: Always use the live GitHub API releases tag crawler when downloading third-party experimental or latest assets to insulate the application from upstream deletion/naming drifts.
- **Official Tauri Plugins for Native UI**: Always prefer standard Tauri-native core plugins (like `tauri-plugin-dialog`) for file/folder pickers rather than scripting fallbacks. This ensures optimal visual styling, modern OS integration, and dynamic front-end customization of file extensions/folders.
